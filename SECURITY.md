# Security Policy

## Supported Versions

Version 1.0.0 and above are supported, older ones are not.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| 0.x.x   | :x:                |

## Reporting a Vulnerability

If you hit a security vulnerability, please inform me directly via email at katajakasa@gmail.com !
